1. Introduction & Endogeneity

เมื่อ OLS ให้ผลลัพธ์ที่ไม่สอดคล้อง (inconsistent)

ปัญหา Endogeneity คืออะไร?

OLS Assumption: ตัวแปรอิสระ x ไม่สัมพันธ์กับ error term u

E[u | x] = 0 \implies \text{Cov}(x, u) = 0

Endogeneity: เมื่อ Cov(x, u) ≠ 0 → OLS ให้ค่า inconsistent

สาเหตุหลัก:

  • Omitted variables — ตัวแปรที่ตกหล่น
  • Measurement error — วัดตัวแปรผิดพลาด
  • Simultaneity — x กับ y กำหนดกันและกัน

ตัวอย่าง: ประกันสุขภาพ & ค่ายา

ldrugexp = f(hi_empunion, totchr, age, female, ...)

hi_empunion (ประกันสุขภาพจากนายจ้าง/สหภาพ) เป็น endogenous

เพราะคนที่เลือกซื้อประกันอาจมี unobserved health status ต่างกัน → Cov(hi_empunion, u) ≠ 0

ผลของ Endogeneity ต่อ OLS

Asymptotic Bias ของ OLS

\text{plim}(\hat{\beta}_{OLS}) = \beta + \frac{\text{Cov}(x, u)}{\text{Var}(x)}

เมื่อ Cov(x, u) ≠ 0 ค่า OLS จะเบี่ยงเบนไม่ว่าจะเพิ่มขนาดตัวอย่างเท่าไร

ทิศทาง bias:

  • Cov(x, u) > 0 → OLS เกินจริง (overestimate)
  • Cov(x, u) < 0 → OLS ต่ำกว่าจริง (underestimate)

  • นายจ้างที่ให้ประกันดีอาจจ่ายเงินเดือนสูง (omitted variable: สวัสดิการ)

ทางออก: ใช้ Instrumental Variables (IV) เพื่อแก้ปัญหา endogeneity

Quiz: Endogeneity

คำถาม: ข้อใดต่อไปนี้ ไม่ใช่ สาเหตุของ endogeneity?

A. Omitted variable bias — ตัวแปรสำคัญถูกละเว้นจากสมการ
B. Simultaneity — ตัวแปร x กับ y กำหนดค่ากันและกัน
C. Heteroskedasticity — ความแปรปรวนของ error ไม่คงที่
D. Measurement error — วัดค่าตัวแปรอิสระผิดพลาด
ถูกต้อง! Heteroskedasticity ทำให้ OLS ไม่มีประสิทธิภาพ (inefficient) แต่ยังคง consistent ไม่ทำให้เกิด endogeneity

2. IV Theory & 2SLS

ตัวแปรเครื่องมือ (Instrumental Variables) แก้ปัญหา endogeneity อย่างไร

แนวคิด Instrumental Variables

เครื่องมือ (Instrument) z ต้องมีคุณสมบัติ 2 ข้อ

  1. Relevance: Cov(z, x) ≠ 0
    z ต้องสัมพันธ์กับตัวแปร endogenous x
  2. Exogeneity: Cov(z, u) = 0
    z ต้องไม่สัมพันธ์กับ error term u

Exclusion Restriction

z ส่งผลต่อ y เฉพาะผ่าน x เท่านั้น ไม่มีผลตรง

z \rightarrow x \rightarrow y \quad (\text{แต่} \; z \not\rightarrow y \; \text{โดยตรง})

Instruments ในข้อมูล MEPS

  • ssiratio — Social Security income ratio
    สัมพันธ์กับการมีประกัน แต่ไม่ควรกระทบค่ายาโดยตรง
  • multlc — Multiple chronic conditions indicator
    สัมพันธ์กับการเลือกซื้อประกัน

จำนวนเครื่องมือ vs. ตัวแปร Endogenous

  • Just-identified: จำนวน z = จำนวน endogenous x
  • Overidentified: จำนวน z > จำนวน endogenous x

IV Estimator: สูตรพื้นฐาน

Simple IV (Just-identified case)

\hat{\beta}_{IV} = \frac{\text{Cov}(z, y)}{\text{Cov}(z, x)} = \frac{\sum (z_i - \bar{z})(y_i - \bar{y})}{\sum (z_i - \bar{z})(x_i - \bar{x})}

เมื่อมีตัวแปร endogenous 1 ตัว และ instrument 1 ตัว

คุณสมบัติ

  • Consistent: plim(β̂IV) = β (เมื่อ Cov(z,u)=0)
  • Asymptotically normal
  • มี variance สูงกว่า OLS เสมอ — "ต้นทุน" ของการแก้ endogeneity

Variance ของ IV

\text{Var}(\hat{\beta}_{IV}) \approx \frac{\sigma^2}{n \cdot \text{Var}(x) \cdot R^2_{x,z}}

เมื่อ R2x,z ต่ำ (weak instrument) → variance จะสูงมาก

Two-Stage Least Squares (2SLS)

ขั้นตอนที่ 1: First Stage

Regress ตัวแปร endogenous x บน instruments z และ exogenous regressors

x_i = \pi_0 + \pi_1 z_{1i} + \pi_2 z_{2i} + \gamma' w_i + v_i

ได้ค่าพยากรณ์ x̂ ที่ "สะอาด" จาก endogeneity

ขั้นตอนที่ 2: Second Stage

Regress y บน x̂ แทน x จริง

y_i = \beta_0 + \beta_1 \hat{x}_i + \delta' w_i + \epsilon_i

ค่า β̂2SLS จะ consistent เมื่อ instruments ถูกต้อง

ลองดู: 2SLS ทีละขั้นตอน

First stage: F-stat แสดงความแข็งแรงของ instrument

2SLS ใน Stata: Just-identified

เครื่องมือ 1 ตัว (ssiratio) สำหรับตัวแปร endogenous 1 ตัว (hi_empunion)

* Just-identified 2SLS with one instrument
ivregress 2sls ldrugexp (hi_empunion = ssiratio) totchr age female blhisp linc, vce(robust) first

ตัวเลือก first

แสดงผล first-stage regression → ดู F-statistic ว่า instrument ssiratio มีพลังพอหรือไม่

Syntax ของ ivregress

ivregress 2sls y (endog = instruments) exog_vars

  • ตัวแปรใน ( ) คือ endogenous
  • หลังเครื่องหมาย = คือ excluded instruments
  • ตัวแปรนอก ( ) คือ included exogenous

2SLS: Overidentified

เครื่องมือ 2 ตัว (ssiratio, multlc) สำหรับตัวแปร endogenous 1 ตัว → overidentified

* Overidentified 2SLS with two instruments
ivregress 2sls ldrugexp (hi_empunion = ssiratio multlc) totchr age female blhisp linc, vce(robust)

ข้อดีของ Overidentification

  • ประสิทธิภาพสูงขึ้น (lower variance)
  • ทดสอบ instrument validity ได้ (overid test)
  • เลือก estimator ได้หลากหลาย (GMM, LIML)

เปรียบเทียบ OLS vs 2SLS

สัมประสิทธิ์ hi_empunion จาก IV มักแตกต่างจาก OLS อย่างมาก

ถ้าต่างกันมาก → สัญญาณว่า endogeneity เป็นปัญหาจริง

3. IV Estimators

GMM, LIML, JIVE — ทางเลือกนอกเหนือจาก 2SLS

GMM: Generalized Method of Moments

หลักการ GMM

เมื่อ overidentified → มี moment conditions มากกว่าพารามิเตอร์

E[z_i \cdot u_i] = 0 \quad (\text{moment conditions})

GMM หาค่า β ที่ทำให้ sample moments ใกล้ศูนย์ที่สุด (weighted)

ข้อดีของ GMM

  • Efficient กว่า 2SLS เมื่อมี heteroskedasticity
  • ใช้ optimal weighting matrix ให้น้ำหนัก moment conditions
  • 2SLS เป็นกรณีพิเศษของ GMM (identity weight)
* GMM with robust weighting matrix
ivregress gmm ldrugexp (hi_empunion = ssiratio multlc) totchr age female blhisp linc, wmatrix(robust)
* Iterated GMM
ivregress gmm ldrugexp (hi_empunion = ssiratio multlc) totchr age female blhisp linc, wmatrix(robust) igmm
* GMM with clustered weighting matrix
ivregress gmm ldrugexp (hi_empunion = ssiratio multlc) totchr age female blhisp linc, wmatrix(cluster age)

igmm: ทำซ้ำจนกว่า weighting matrix จะ converge → ประสิทธิภาพสูงสุด

LIML & JIVE

LIML (Limited Information Maximum Likelihood)

Likelihood-based estimator ที่มีข้อดีเมื่อ instruments อ่อน

  • Median-unbiased (approximate) — bias น้อยกว่า 2SLS
  • ทนต่อ weak instruments ดีกว่า 2SLS
  • แต่อาจมี fat tails (moments อาจไม่มี)
* LIML estimation
ivregress liml ldrugexp (hi_empunion = ssiratio multlc) totchr age female blhisp linc, vce(robust)

JIVE (Jackknife IV Estimator)

ใช้ jackknife ในการสร้าง fitted values ของ first stage

  • แก้ bias จาก overfitting ใน first stage
  • ดีกว่า 2SLS เมื่อมี many instruments
  • Consistent แม้จำนวน instruments จะเพิ่มตาม n
* JIVE estimation (user-written command)
jive ldrugexp (hi_empunion = ssiratio lowincome multlc firmsz) totchr age female blhisp linc, robust

เปรียบเทียบ IV Estimators

คำถาม: เมื่อมี heteroskedasticity และ instruments แข็งแรง ควรใช้ estimator ใด?

A. OLS — เพราะมีประสิทธิภาพสูงสุดเสมอ

4. Testing

First Stage, Endogeneity Test, Overidentification Test

First-Stage Regression & Instrument Relevance

ทำไมต้องดู First Stage?

ถ้า instruments ไม่สัมพันธ์กับ endogenous variable → IV จะไม่ทำงาน

x_i = \pi_0 + \pi_1 z_{1i} + \pi_2 z_{2i} + \gamma' w_i + v_i

ทดสอบ: H0: π1 = π2 = 0

Rule of Thumb: F > 10

Staiger & Stock (1997): ถ้า first-stage F-statistic < 10 → weak instruments

Stock-Yogo (2005): ให้ critical values ที่เข้มงวดกว่า

* After ivregress, check first stage
estat firststage, forcenonrobust all

ผลลัพธ์ที่ต้องดู

  • F-statistic — ยิ่งสูงยิ่งดี (> 10)
  • Partial R2 — ส่วนที่ instruments อธิบาย x ได้
  • Minimum eigenvalue — ใช้เทียบกับ Stock-Yogo critical values
* Correlation of instrument with endogenous
correlate hi_empunion ssiratio lowincome multlc firmsz if linc!=.

Endogeneity Test (Durbin-Wu-Hausman)

คำถาม: x เป็น endogenous จริงหรือ?

ถ้า x ไม่ endogenous → OLS ดีกว่า (efficient กว่า IV)

H_0: \text{Cov}(x, u) = 0 \quad (\text{exogenous})
H_1: \text{Cov}(x, u) \neq 0 \quad (\text{endogenous})

ถ้า reject H0: x เป็น endogenous จริง → ใช้ IV

ถ้า fail to reject: ไม่มีหลักฐานว่า endogenous → OLS อาจเพียงพอ

* Durbin-Wu-Hausman test (automatic)
ivregress 2sls ldrugexp (hi_empunion = ssiratio) totchr age female blhisp linc, vce(robust)
estat endogenous
* Manual DWH test
quietly regress hi_empunion ssiratio totchr age female blhisp linc
quietly predict v1hat, resid
quietly regress ldrugexp hi_empunion v1hat totchr age female blhisp linc, vce(robust)
test v1hat

วิธี manual: เพิ่ม residuals จาก first stage เข้าไปใน OLS ถ้า v1hat มีนัยสำคัญ → x เป็น endogenous

Overidentification Test (Sargan/Hansen J)

คำถาม: Instruments ทั้งหมด valid หรือไม่?

ทดสอบได้เฉพาะเมื่อ overidentified (instruments > endogenous)

H_0: \text{All instruments are valid}
J = n \cdot \hat{u}' Z (Z'Z)^{-1} Z' \hat{u} \sim \chi^2_{q-k}

q = จำนวน instruments, k = จำนวน endogenous

ถ้า reject H0: อย่างน้อย 1 instrument ไม่ valid → ทบทวน

ถ้า fail to reject: ไม่มีหลักฐานว่า instruments ไม่ดี

* Overidentification test after GMM
ivregress gmm ldrugexp (hi_empunion = ssiratio multlc) totchr age female blhisp linc, wmatrix(robust)
estat overid
* With 4 instruments
ivregress gmm ldrugexp (hi_empunion = ssiratio lowincome multlc firmsz) totchr age female blhisp linc, wmatrix(robust)
estat overid

ข้อจำกัด

J-test ทดสอบได้เฉพาะว่า instruments ทั้งหมด ไม่สอดคล้องกันหรือไม่

ถ้า instruments ทั้งหมดไม่ valid ในทิศทางเดียวกัน → J-test อาจไม่ detect

Overidentification: Intuition

ลองดู: J-test ทำงานอย่างไร?

เมื่อมี 2 instruments แต่ละตัวให้ค่า IV estimate ต่างกัน ถ้าทั้งสอง valid → ค่าใกล้กัน ถ้าตัวใดตัวหนึ่งไม่ valid → ค่าแตกต่างกันมาก

J-test: Instruments ทั้งสองให้ค่าใกล้กัน → fail to reject H0

คำถาม: Overidentification test (J-test) ทดสอบสมมติฐานใด?

ปัญหา Weak Instruments

Weak Instruments คืออะไร?

เมื่อ instruments สัมพันธ์กับ endogenous variable เพียงเล็กน้อย

  • First-stage F-statistic ต่ำ (< 10)
  • Partial R2 ต่ำมาก

ผลเสีย

  • 2SLS จะ biased toward OLS
  • Standard errors จะ misleading (เล็กเกินจริง)
  • Confidence intervals ผิดพลาด
  • t-test มี size distortion มาก

Checklist การวิเคราะห์ IV

  1. First-stage F > 10? → instruments แข็งแรง
  2. DWH test reject? → x เป็น endogenous จริง
  3. J-test fail to reject? → instruments valid
  4. ผลลัพธ์ stable ข้าม estimators? → น่าเชื่อถือ

Quiz: Weak Instruments

คำถาม: ถ้า first-stage F-statistic = 3 ควรทำอย่างไร?

A. ใช้ 2SLS ต่อไปได้ เพราะ IV ยัง consistent
B. ใช้ OLS แทน เพราะ IV ไม่ทำงาน
C. เปลี่ยนไปใช้ LIML หรือ JIVE ที่ทนต่อ weak instruments ได้ดีกว่า หรือหา instruments ที่ดีกว่า
D. เพิ่ม instruments ให้มากขึ้นเพื่อเพิ่ม F-statistic
ถูกต้อง! เมื่อ F < 10, 2SLS จะ biased toward OLS LIML/JIVE มี bias น้อยกว่า แต่ทางออกที่ดีที่สุดคือหา instruments ที่แข็งแรงกว่า การเพิ่มจำนวน instruments อ่อนๆ อาจทำให้ bias แย่ลง

6. Summary

สรุปเนื้อหา Chapter 4: Instrumental Variables

สรุป: IV Estimation

ปัญหา

  • Endogeneity ทำให้ OLS inconsistent
  • สาเหตุ: omitted variables, simultaneity, measurement error
  • Bias ไม่หายไปแม้เพิ่ม n

ทางออก: IV

  • หา instrument z ที่ relevant & exogenous
  • 2SLS, GMM, LIML, JIVE
  • Trade-off: consistency vs. efficiency

การทดสอบ

  • First-stage F > 10 (relevance)
  • DWH test (endogeneity)
  • J-test (overid/validity)
  • Stock-Yogo (weak IV)

Key Takeaways:

  • IV แก้ปัญหา endogeneity แต่ต้อง "จ่าย" ด้วย variance ที่สูงขึ้น
  • Instrument ที่ดีหายาก — ต้องมีทั้ง relevance และ exogeneity
  • ทดสอบ diagnostic tests ทุกครั้ง: first stage, endogeneity, overid
  • ถ้า instruments อ่อน → ใช้ LIML/JIVE หรือหา instruments ใหม่
